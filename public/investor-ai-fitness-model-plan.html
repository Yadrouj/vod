پیشنهاد فنی مدل محلی

لایه انتخاب پیشنهادی چرا نقش در MVP
مدل اصلی چندوجهی Qwen3-Omni-30B-A3B-Instruct / Thinking طبق گزارش Qwen، ورودی متن، تصویر، صوت و ویدیو را پشتیبانی می کند و خروجی متن و گفتار می دهد؛ برای یک دستیار فیتنس فارسی چندوجهی بهترین گزینه متن باز عملی است. پاسخ هوشمند، فهم تصویر، تحلیل صدای کاربر، توضیح حرکت و گزارش بدن.
مدل سبک موبایل/لبه Gemma 3n Google آن را برای اجرای کم مصرف و ورودی متن، تصویر، صدا و ویدیو معرفی کرده است. برای نسخه آفلاین سبک یا تشخیص کیفیت تصویر مناسب است. تشخیص اولیه کیفیت عکس، چت سریع، fallback روی دستگاه یا سرور ارزان.
لایه سلامت و احتیاط پزشکی MedGemma 1.5 4B / 27B برای فهم متن و تصویر پزشکی آموزش دیده، اما باید فقط پس از اعتبارسنجی و با محدودیت ایمنی استفاده شود. تشخیص red flag، ارجاع به پزشک، جلوگیری از توصیه خطرناک درباره دارو، آسیب، درد قفسه سینه، بارداری و بیماری زمینه ای.
مدل چت سریع فارسی Qwen3 14B / 32B LoRA + RAG برای هزینه پایین تر، پاسخ سریع تر و فاین تیون اختصاصی روی دیتاست فارسی فیتنس. ۹۰٪ سوال های روزمره: تمرین، تغذیه، مکمل، برنامه هفتگی، FAQ و مقالات.
صدا Persian ASR + TTS, then Qwen Omni speech برای MVP کم ریسک تر است که صدا به متن، سپس مدل متنی، سپس گفتار فارسی تولید شود. بعدا Qwen Omni می تواند مستقیم تر شود. مربی صوتی، پرسش سریع در تمرین، پاسخ خواندنی و شنیدنی.
تحلیل بدن از عکس BodyM/SMPL pipeline + VLM review برای تخمین بدن نباید فقط به VLM عمومی تکیه کرد. باید از pose، segmentation، silhouette، قد/وزن و مدل اندازه گیری استفاده شود. کیفیت عکس، تخمین دورها، روند تغییر بدن، توصیه تمرین و تغذیه؛ نه تشخیص پزشکی قطعی.

VGA و سخت افزار فاین تیون

اصل تصمیم

برای MVP نباید از صفر pretrain کنیم. مسیر اقتصادی و قابل دفاع: RAG + LoRA/QLoRA + ارزیابی اختصاصی + داده مالکیتی. QLoRA در مقاله اصلی نشان داده که می توان مدل ۶۵B را با ۴ بیت روی یک GPU ۴۸GB فاین تیون کرد؛ اما برای مدل های چندوجهی، batch، طول کانتکست و تصویر، حافظه بیشتری لازم می شود.

سناریو GPU پیشنهادی قابل انجام محدودیت
شروع اقتصادی 1x RTX 5090 32GB LoRA مدل های 4B تا 14B، inference مدل های quantized، ساخت دیتاست و eval. برای فاین تیون جدی Qwen3-Omni 30B چندوجهی کم حافظه است.
MVP جدی پیشنهادی 1x RTX PRO 6000 Blackwell 96GB QLoRA مدل های 27B/30B، تست multimodal، inference پایدار، batch بهتر. قیمت وارداتی و تامین در ایران باید روزانه استعلام شود.
جایگزین در دسترس تر 2x RTX 6000 Ada 48GB تقسیم inference، LoRA مدل های 30B/70B quantized، پایداری workstation. نسل Ada قدیمی تر از Blackwell است؛ سرعت/حافظه هر کارت کمتر از 96GB است.
لابراتوار سرمایه گذار 4x RTX PRO 6000 Blackwell 96GB فاین تیون های موازی، دیتاست بزرگ تر، experiment سریع، چند سرویس inference. نیازمند کیس، پاور، کولینگ، برق پایدار و MLOps جدی.
تحقیق سنگین/Full fine-tune Cloud H100/H200 cluster Full fine-tune یا pretraining محدود، jobهای چندروزه، مقیاس صنعتی. برای ۱۰ میلیارد تومان فقط باید اجاره هدفمند و کوتاه مدت باشد، نه هزینه دائمی.
